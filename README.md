# SafeHer – Women Safety Ride Booking Platform - Niharika-2211985035-copyright

SafeHer is a full-stack web application designed to provide a safer ride-booking experience for women. The platform offers women-only taxi services with verified female drivers, live ride tracking, emergency SOS support, ride history, and secure authentication.

## Features

### User Features
- Secure Login / Signup
- Book Safe Rides
- Live Ride Tracking
- Emergency SOS Button
- Favorite Locations
- Ride History
- User Profile Management
- Feedback / Ratings

### Safety Features
- Verified Female Drivers
- Real-Time Location Sharing
- Emergency Contacts
- One-Tap SOS Alerts
- 24/7 Safety Monitoring

## Tech Stack
